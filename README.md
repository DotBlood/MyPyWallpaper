# Wallpaper 



## Запуск

Для IDE:
  ``` 
  1. python3 -m venv ./venv
  2. cd ./venv/Scripts/ 
  3. .\activate
  4. -m pip install -U autopep8 
  5. python3 main.pyw
  ```

Для запуска простого:
  ```
 * bootstrapper.bat
  ```

Для exe:
 ``` 
 1. MyPyWallpaper.exe
 * Eсли вы хотите добавить программу в автозагрузку, запустите bootstrapper.bat в котологе программы.
  ```
